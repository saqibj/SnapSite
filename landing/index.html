How it works

Three steps from “I need captures” to a folder full of evidence—no SaaS login required.

Load the extension

Clone the repo and load the unpacked build in Chrome—see the install guide for the exact clicks.

Configure the crawl

Paste a start URL, tune depth and delays, and add exclude patterns for admin, PDFs, or assets.

Run & collect PNGs

Start the crawl, pause if needed, and let SnapSite drop full-page screenshots into Downloads.
